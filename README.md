# IL (ice logger)

Is a CLI Logger thing to save `.log` files fast directly from the Terminal.

## How does it work?

Use `il "{log_text} {log_name}"` to create a log. <br>
Use an Option `savmult` (Save Multiplier) to save more `.log` files (it doesn't even change them?? lmaoo) <br>
Use a Flag `copy` to copy log's data.

## Requirements

macOS 12 or higher
